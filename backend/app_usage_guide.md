## CHUNK: How to use the Landed app
### Overview
**category:** reference
**personas:** all
**is_free:** true

Landed helps you find trusted NYC resources and guidance across food, housing, health, legal rights, school, transit, and work.  
Main screens: Home, Ask, Map, Saved, and Profile.

---

## CHUNK: Home screen and category guides
### Home basics
**category:** reference
**personas:** all
**is_free:** true

Use Home to browse support categories and open step-by-step guides.

- Open a category card (Food, Health, Housing, etc.) to view practical instructions.
- Use the “More” option to see additional categories.
- Each guide includes what to do now, key tips, and where to get official help.

---

## CHUNK: Ask screen (chatbot)
### Ask chatbot usage
**category:** reference
**personas:** all
**is_free:** true

Use Ask to get grounded answers from trusted resources in the app.

- Type your question in plain language (example: “I need food help in Queens today”).
- The chatbot returns:
  - direct answer
  - citations (sources)
  - resource cards with links
  - action buttons (open guide/map/resource/call)
- If a link appears, tap it to open official information.
- If you need urgent help, ask for “emergency options” and call listed hotlines.

**category:** reference
**personas:** all

## CHUNK: Common user tasks
### Quick tasks and where to go
**category:** reference
**personas:** all
**is_free:** true

- “I need food now” → Ask or Food guide, then Map for nearby pantries.
- “I need a clinic” → Ask or Health guide, then Map.
- “I need housing help” → Housing guide + Ask for next steps.
- “I want legal rights info” → Legal guide + Ask for official contacts.
- “How do I change language?” → Profile.

## CHUNK: Troubleshooting chatbot
### If Ask feels wrong or incomplete
**category:** reference
**personas:** all
**is_free:** true

If chatbot output is unclear:

- Ask a more specific question (need + location + urgency).
- Ask for “official links only.”
- Ask “what documents do I need?”
- If urgent/safety issue, use hotline actions immediately.
